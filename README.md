# 🛍️ Myntra Clone (Frontend Project)

This is a Myntra-inspired e-commerce website built using HTML & CSS.  
The goal of this project was to practice real-world UI design and layout structuring.

---

## 🚀 Features

- Multi-level navigation menu
- Category page with filters & sorting (CSS-based)
- Product detail page design
- Clean and modern UI
- Hover interactions without JavaScript

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Positioning)

---

## 📂 Project Structure

- index.html
- category.html
- product.html
- css/
- Images/

---

- Advanced CSS layouts
- Real-world UI structuring
- Navigation design like e-commerce websites
